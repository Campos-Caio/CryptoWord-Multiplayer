Bem-vindo ao CryptoWord Multiplayer!

Regras do Jogo:

  • 1. Escolha seu nickname e conecte-se.
  • 2. Aguarde o mínimo de 2 jogadores para iniciar.
  • 3. Clique nas letras no grid para formar palavras.
  • 4. As letras devem ser adjacentes (horizontal/vertical).
  • 5. Envie a palavra selecionada para pontuar.
  • 6. Encontre todas as palavras para vencer a rodada!
